Documentation for Quadrocopter Navigation code : 

->This code makes use of TLD to track objects and get feedback on their location.
->The function "guidance()" continously keeps on finding the next waypoint.
->The function "get_guidance()" returns a pointer to an array. The array stores the x,y,z coordinates of the next waypoint.

Example showing use of the guidance function in a sample c++ class:

#include <TLD/include/central.h>
#include <io.h>

void main(string args[])
{
	
	while(true)
	{
		double *arr=get_guidance();
		cout <<"Next waypoint : "<<arr[0]<<"\t"<<arr[1]<<"\t"<<arr[2]<<"\n";
		//Condition to break out of the loop if required....
	}
}
